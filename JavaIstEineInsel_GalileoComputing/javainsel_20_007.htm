20.7 Zum Weiterlesen 

Besonders die Java 2D-API kann aber noch vieles mehr, was hier nicht weiter erwähnt wurde. Dazu zählen Bildfilter, etwa zum Weichzeichnen, die im Java 2D-Demo des Java JDK vorgestellt werden. Zusammen mit der »Binding for the OpenGL API« (JSR-231) über JOGL (https://jogl.dev.java.net/) steht auch anspruchsvollen performanten Spielen nichts mehr im Wege.
